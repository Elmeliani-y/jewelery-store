## Admin Secret Link

- **Format:** `http://your-domain/7xK9mP2vQ8nL4wR6tY3zA5bN1cJ0hF8e`
- **Length:** 32 characters
- **Functionality:** Direct admin access, bypasses IP blocks

## Environment Variables

- `APP_URL_PREFIX`: Default is `xK9wR2vP8nL4tY6zA5bM3cH0jG7eF1dQ`
- All application routes are prefixed with this value

## Security Features

1. All routes are obfuscated with random alphanumeric strings
2. Root domain shows blank landing page
3. Invalid routes show blank page (no 404 exposure)
4. Device tokens are 40 characters
5. Admin secret is 32 characters
6. IP-based blocking and tracking
7. First-use IP binding for device tokens
